# Slovo

Slovo — это voice-first приложение для общения, построенное вокруг серверов и живых голосовых каналов. Оно рассчитано на группы, сообщества и команды, которым нужно быстро собираться, видеть, кто активен прямо сейчас, подключаться к общему голосовому пространству и координироваться через демонстрацию экрана без text-first сценария.

Проект решает задачу легкой коммуникации в реальном времени с постоянной структурой серверов и понятными границами владения. Он сочетает постоянные данные сервера и membership с live presence, голосовыми сессиями, приглашениями, модерацией и screen share, чтобы пользователь мог быстро перемещаться между пространствами, а owner сохранял контроль над доступом и конфигурацией.
